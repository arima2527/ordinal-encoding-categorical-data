# Ordinal Encoding for Categorical Data

## 📌 Project Overview
This project demonstrates how to convert categorical data with a natural order into numerical form using Ordinal Encoding. Categories are assigned numerical values based on their order or ranking.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 📊 Features
- Identifies ordinal categorical columns
- Applies Ordinal Encoding
- Converts ordered categories into numerical values
- Prepares dataset for machine learning models

## 🔧 Method Used
Ordinal Encoding:
- Categories with a natural order are converted into numbers.
- Each category is assigned a numerical value according to its rank.

Example:

Category | Encoded Value  
Low | 0  
Medium | 1  
High | 2  

## ⚙️ How to Run the Project
1. Clone or download the repository.
2. Open the Jupyter Notebook file.
3. Run the notebook cells step by step.
4. The notebook will demonstrate Ordinal Encoding on the dataset.

## 🎯 Purpose
Some categorical variables have a natural order (for example: Low, Medium, High). Ordinal Encoding helps convert these ordered categories into numerical values so that machine learning models can process them effectively.
